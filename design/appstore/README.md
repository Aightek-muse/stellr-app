## Upload Instructions

### App Store Connect

1. Go to **My Apps** → **Stellr** → **App Store** → **Versions**
2. Scroll to **App Preview** section
3. Upload screenshots for each display size:
   - **6.7"** (required) — upload all 7 from `6.7/` folder
   - **6.5"** (required) — upload all 7 from `6.5/` folder
   - **5.5"** (required) — upload all 7 from `5.5/` folder

### Recommended Order

1. Welcome (hero hook)
2. Question (shows the unique mechanic)
3. Moon Reveal (emotional peak — the screenshot moment)
4. Summary (social/share moment)
5. Home Dashboard (habit loop)
6. Sun Reveal (optional — shows depth)
7. Paywall (optional — shows premium value)
